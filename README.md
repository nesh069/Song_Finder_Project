# Song_Finder_Project
A website that helps users search for songs by title, artist, genre or popularity.

---

## What it does 

- Search for songs in real time 

- Filter by genre

- Sort by popularity

- Built with plain HTML, Vanilla CSS and JavaScript

---

## Project Objectives 

- Use a a public API for song data 

- Responsive on mobile and desktop

- Maximum 3 colors in design

- Three different fot families

- Unit tests fot the search and filter logic

- At least 20 commits with descriptive messages

- Deployed with GitHub actions to GitHub pages 

- Vanilla CSS only

---

## Live Site

[https://nesh069.github.io/Song_Finder_Project](https://nesh069.github.io/Song_Finder_Project)

## How to run it

Just open `index.html` in your browser. No setup needed.

To run tests:

```bash
npm install
npm test
```
