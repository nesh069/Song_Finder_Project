# Song_Finder_Project

A website that helps users search for songs by title, artist, genre or popularity.

---

## What it does

- Search for songs in real time
- Filter by genre
- Sort by popularity
- Discover music by region (African, Latin, Asian, Brazilian, Indian, Reggae)
- Click any song card to see album details
- Play a 30-second preview of any song
- Built with plain HTML, Vanilla CSS and JavaScript

---

## Project Objectives

- Use a public API for song data
- Responsive on mobile and desktop
- Maximum 3 colors in design
- Three different font families
- Unit tests for the search and filter logic
- At least 20 commits with descriptive messages
- Deployed with GitHub Actions to GitHub Pages
- Vanilla CSS only

---

## APIs Used

- **Last.fm API** — song search and popularity data
- **Deezer API** — album covers, audio previews and track details
